### Description

A simple portfolio  built with React.js and email.js


### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap
- Edit Content From One Place

### Setup
 
Install required dependencies

<pre>yarn install</pre>


Start the server

<pre>yarn start</pre>
